🇮🇳 Seva Setu Widget

Consulate General of India — Johannesburg
Live demo. The chat widget is loaded below via a single <script> tag.

Embed on any website — copy this 1 line:

<script src="https://sevasetu.seva.org.za/seva-widget.js" data-bot-id="1"></script>

Replace YOUR-HOST.com/seva-widget.js with wherever you host the built file (dist/widget/seva-widget.js after running yarn build:widget). The backend URL is baked in at build time via REACT_APP_BACKEND_URL.
